JavaScript Cheatsheet

Variables
  • var - older way of declaring a variable (use let instead)
  • let - variable declaration of which its value CAN be changed
  • const - variable declaration of which its value CANNOT be changed
Data Types

Primitive data types can only store one thing; however, objects store a collection of data so they are NOT primitive. You can use typeof x or typeof(x) to find which type of data has been stored.

  1. number: numbers of any kind: integor or floating point
  2. bigint: integer numbers of arbitrary length
  3. string: a string of one of more characters
  4. boolean: true/false
  5. null: unknown values
  6. undefined: unassigned values
  7. object: complex data structures
  8. symbol: unique identifiers

You can convert types using the following:

String(value); // string conversion
Number(value); // numeric conversion
  /* undefined ~> NaN
  * null ~> 0
  * true / false ~> 1 / 0
  * string ~> error gives NaN, empty string gives 0 
  */
Boolean(value); // boolean conversion
  /* 0, null, undefined, NaN, "" ~> false (aka "falsey")
  * anything else ~> true (aka "truthy")
  */
Operators

+ (binary): adds numbers and if operands are strings, will concatenate strings

+ (unary): converts non-numbers into numbers (+true ~> 1 OR +"" ~> 0 OR +"2" ~> 2). example:

let apples = "2";
let oranges = "3";
// both values converted to numbers before the binary plus
alert( +apples + +oranges ); // 5

=: assignment

%: modulus (remainder)

**: exponentiation, example:

console.log(2 ** 2); // 4  (2 * 2)
console.log(2 ** 3); // 8  (2 * 2 * 2)
console.log(2 ** 4); // 16 (2 * 2 * 2 * 2)
console.log(8 ** (1/2)); // 2 (power of 1/2 is the same as square root)

++: increment variable by 1, example:

++counter; // increase a value AND immediately use the result of the operator
counter++; // increase a value but use its previous value

--: decrement variable by 1

+=, -=, *=, /=: modify-and-assign (n = n + 5)

(...): grouping operator; operations within brackers are executed earlier

if Statements
if (condition) {
  // do stuff
} else if (condition) {
  // do this stuff instead
} else {
  // if conditions above aren't meant, do this stuff
}

Switch Statements

// switch statements are always strict!
let arg = prompt("Enter a value?");
switch (arg) {
  case '0':
  case '1':
    alert( 'One or zero' );
    break;

  case '2':
    alert( 'Two' );
    break;

  case 3:
    alert( 'Never executes!' ); // because it's not a string '3'
    break;
  default:
    alert( 'An unknown value' );
}

Comparison Operators

  • ==: regular equality operator
  • ===: strict equality operators
  • !=: not equal
  • !==: not equal value or not equal type
  • >: greater than
  • <: less than
  • >=: greater than or equal to
  • <=: less than or equal to
  • ? (ternary): conditional operator
Logical Operators

|| (logical or)

The || (logical or) evaluates operands from left to right. Each operand is converted to boolean type and if TRUE, it stops and returns the value of that operand. If all operands have been evaluated (false), then it returns the last operand.

&& (logical and)

The && (logical and) evaluates operands from left to right. Each operand is converted to boolean type and if result is FALSE, it stops and returns the value of that operand. If all operands have been evaluated (true), then it returns the last operand.

! (logical not)

The ! (logical not) converts the operand to boolean type. It returns the inverse value. !! is sometimes used to convert a value to boolean type as well. The more verbose version of !!value is Boolean(value);
